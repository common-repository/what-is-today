今日は何の日？ふっふー

== Description ==

<?php echo what_is_today(); ?>を書いておくだけで今日が何の日か分かります。

== Installation ==

1. Upload `what-is-today.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php echo what_is_today(); ?>` in your templates
